## If you are ready to build and release a version

- run `make build` to build the lib
- run `npm version` with one of the arguments:
	* `major` => v1.0.0
	* `minor` => v0.1.0
	* `patch` => v0.0.1
- `git push` and `git push --tags`
- optionally, `bower publish` and/or `npm publish`
